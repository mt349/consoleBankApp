# ConsoleBankApp

**Description**  
A console-based banking system implemented in Java.  
It allows users to register, log in, create multiple accounts, and perform deposit/withdrawal operations. Data is persisted using Java serialization.

---

## Features

- User registration and login
- Create multiple bank accounts per user
- Deposit and withdraw funds from accounts
- List all accounts with balances
- Save and load data automatically using serialization

---

## Classes Overview

1. **Bank**
   - Manages all users
   - Handles registration, login, saving/loading data

2. **User**
   - Stores individual user information
   - Manages accounts (add, list, interact)

3. **Account**
   - Stores account number and balance
   - Handles deposit and withdraw operations

4. **Main**
   - Console-based interface
   - Handles user input and calls Bank/User methods
